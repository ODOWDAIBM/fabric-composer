Class: ModelManager

ModelManager

The ModelManager manages a set of ModelFiles. Each ModelFile has a single namespace and defines a set of classes (assets, transactions, participants).

new ModelManager()

Create the ModelManager. This should only be called by the framework. Instances should be retrieved from the Concerto module.
