Class: ModelManager

ModelManager

The ModelManager manages a set of Concerto language files. Each Concerto file has a single namespace and defines a set of classes (enums, assets, transactions, participants) and transactions processor functions.

new ModelManager()

Create the ModelManager. This should only be called by the framework. Instances should be retrieved from the Concerto module.
