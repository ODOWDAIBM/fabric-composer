IBM Blockchain Framework

Get Started!

IBM Blockchain Framework is a set of libraries and tools to quickly create blockchain-based applications. 

npm install -g @ibm/concerto-cli
git clone git@github.ibm.com:Blockchain-WW-Labs/Concerto-GettingStarted.git

What is IBM Blockchain Framework?

IBM Blockchain Framework is a set of APIs, a modeling language and a programming model to quickly define and deploy business networks and applications that allow participants to send transactions that exchange assets.

Hyperledger Fabric Open Source technology, is used to store the state of assets in asset registries, while the blockchain consensus protocol ensure that transactions are validated by concerned organizations in the business network.

Framework Features

Blockchain State Storage
All state is stored in Hyperledger Fabric, ensuring consensus, immutability etc.
Assets
Assets & Asset Registries define the data to be stored and exchanged
Particpants
Transactions are associated with authenticated members of the Hyperledger Fabric
REST APIs
Provides REST APIs to interact with assets and the transaction registry
Analytics and Monitoring
Extension points to send transactions to external analytics data stores
